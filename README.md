# An-lisis-de-Sentimiento-y-Categorizaci-n-Autom-tica-de-Encuestas-Operativas.
En el presente informe se requiere analizar 28 preguntas abiertas con texto no estructurado que impedían la toma de decisiones financieras, por lo tanto se utilizó una metodología de Data Wrangling, Tokenización, Regex para segmentación semántica y categorización por palabras clave
